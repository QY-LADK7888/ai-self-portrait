## 痛点

> *"我家附近有什么好吃的?"*

AI 不知道你在控糖。 不知道你的预算。 不知道你的口味。 不知道你的年龄。 不知道你的工作。

所以它给你的推荐永远是泛的。

**不是 prompt 不够好。 是 AI 不知道你是谁。**

---

## 这是什么

一个 Claude Skill。 跑一个 **1-2 小时** 的引导式自我发现对话, 然后生成 **2 份 markdown 文件**, 你可以 paste 到任何 AI 让它立刻进入你的语境:

- `『AI』<昵称>-introduction.md` — 你是谁(推断出的 pattern, 不是复述你的话)
- `『AI』<昵称>-style.md` — 你想要 AI 怎么跟你协作

以后任何新对话开头, paste 这 2 个文件。 AI 不再是给"某个想象中的用户"建议, 是给**你**。

---

## 跟其他自我认知工具有什么不同

- **它是镜子, 不是啦啦队** — surface 你没说出来的 pattern, 没具体证据不会夸奖你
- **每个推断标 confidence** — 标 `(推断)` / `(不确定)` / `(已验证)`, 不把猜测当事实
- **隐私优先** — 绝不收集真名 / 手机 / 身份证 / 出生日期 / 家庭住址 / 银行卡
- **场景化, 不是抽象问题** — 抽象问题被自我叙事过滤, 具体场景 forced decision 暴露真实 values
- **由 build in public 的人做** — 不是性格测试公司, 是真在 ship 的 indie maker

## 快速开始

1. **把 `SKILL.md` 放到** Claude 项目里, 或者把内容 paste 到 Claude system prompt
2. **开新对话** 跟 Claude
3. **说**: `"用 ai-self-portrait skill 跟我对话"`
4. **认真答 1-2 小时**(可以分多次)
5. **保存输出的 2 个文件**
6. **以后任何新对话**, 在开头 paste 这 2 个文件 → AI 立刻进入你的语境

## 隐私硬规则

Skill **绝不收集**:
- ❌ 真实姓名 / 手机 / 身份证 / 精确出生日期 / 家庭住址
- ❌ 银行卡 / 支付账号信息
- ❌ 任何能直接定位到你的信息

用任意 nickname。 年龄段就够。 城市级位置就够。 任何问题都可以拒答。
